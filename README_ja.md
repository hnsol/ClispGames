**カナガンの涙を集め、仲間を救い出せ！**

> 異世界カナガンに迷い込んだ3匹の仲良しネコ、ハンちゃん、ソロ、ナナチ。恐怖で動けなくなったハンちゃんを救うため、兄弟のソロと親友ナナチが奇跡を求めて冒険へ！二つの異なる世界、“灰色の世界”と“黒の世界”を舞台に、２匹はカナガンの涙を集め、仲間を救い出すために奮闘します。それぞれの世界で出会う困難を、勇敢なジャンプアクションで乗り越えろ！ハンちゃんの安否がかかっている...時間はない！
