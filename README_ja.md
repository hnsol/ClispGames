# さくさくゲーム集

[ABA games](http://www.asahi-net.or.jp/~cs8k-cyu/)の[crisp-game-lib](https://github.com/abagames/crisp-game-lib)を使用したミニゲーム集です。


## コレクション

[ティアーズ・オブ・ザ・カナガン（Easy版）](http://hnsol.github.io/CrispGames/?tok_easy)

<a href="http://hnsol.github.io/CrispGames/?tok_easy"><img src="https://cdn-ak.f.st-hatena.com/images/fotolife/m/masatora_bd5/20230930/20230930230557.png" width=100%></a>

**カナガンの涙を集め、仲間を救い出せ！**

> 異世界カナガンに迷い込んだ3匹の仲良しネコ、ハンちゃん、ソロ、ナナチ。恐怖で動けなくなったハンちゃんを救うため、兄弟のソロ（黒ネコ）と親友ナナチ（白ネコ）が奇跡を求めて冒険へ！二つの異なる世界、“灰色の世界”と“黒の世界”を舞台に、２匹はカナガンの涙を集め、仲間を救い出すために奮闘します。それぞれの世界で出会う困難を、勇敢なジャンプアクションで乗り越えろ！ハンちゃんの安否がかかっている...時間はない！
